Фотографии

Наша коллекция фотографий. Чтобы добавить свои фото, замените изображения в разделе ниже.

ИНСТРУКЦИЯ ПО РЕДАКТИРОВАНИЮ:

Чтобы добавить свои фотографии:

  • Замените ссылки в атрибуте style="background-image: url('...')" на пути к вашим изображениям
  • Добавьте или удалите блоки <div class="gallery-item"></div> по необходимости
  • Для оптимального отображения используйте изображения с соотношением сторон 1:1

Видео

Наши видео материалы. Добавляйте свои видео по инструкции ниже.

ИНСТРУКЦИЯ ПО РЕДАКТИРОВАНИЮ:

Чтобы добавить свое видео:

  • Замените src="ваше-видео.mp4" на путь к вашему видеофайлу
  • Убедитесь, что видео в формате MP4 для максимальной совместимости
  • Можно добавить несколько видео, скопировав блок <div class="video-container"></div>

Полезная информация

Здесь вы можете разместить любой текстовый контент. Инструкция по редактированию ниже.

ИНСТРУКЦИЯ ПО РЕДАКТИРОВАНИЮ:

Чтобы добавить свой текст:

  • Замените содержимое внутри блока <div class="text-content"></div>
  • Вы можете использовать любые HTML-теги для форматирования (заголовки, списки, ссылки и т.д.)
  • Для добавления нескольких текстовых блоков скопируйте весь блок <div class="text-content"></div>
